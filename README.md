# aiom

Rusty CLI weather

## Usage

Get the current weather at your current location:

```bash
aiom
Shillong, Meghalaya
T: 21°C
W: 6 km/h
H: 51%
```

Get the current weather at some particular location:

```bash
aiom -l Bengaluru
Bengaluru, Karnataka
T: 26°C
W: 13 km/h
H: 34%
```
